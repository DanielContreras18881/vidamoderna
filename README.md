# La Vida Moderna Videogame Show

## Proyecto

¡Hola moderdonias y moderdonios!

Esto es un proyecto de videojuego basado en el programa de radio de La Vida Moderna, con David Broncano, Quequé e Ignatius.
La idea es hacer un proyecto colaborativo en el que todo el quiera pueda aportar. El videojuego base está creado en Unity (5.6.3p3)que es descargable de forma gratuita. De momento puedo subir el juego a mi servidor (www.valnargames.net), y se podría subir tambien a itch.io.
Aún así sois libres de crear un proyecto diferente con otras herramientas, u otro diferente con Unity.

## Flow de trabajo

El repositorio es publico, por lo que deberíais poder usarlo sin problemas. Si tenéis alguna duda, consulta, problema, whatever...podéis contactar conmigo en moonspell.doom@gmail.com
